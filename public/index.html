URL Shortener Microservice

USAGE:

  1. Pass a URL as a parameter using the /getnewurl/ path, the API will check if the URL is valid.
  2. If it is, it will create a shortened URL and return it in a JSON object. Visiting the shortened URL will redirect to the original link.
  3. If it isn't, the JSON response will contain an "Invalid URL" error message.

EXAMPLES:

  • Visiting https://fabvit86-urlshort-ms.herokuapp.com/getnewurl/http://www.nba.com will return {"original_url":"http://www.nba.com", "short_url":"https://fabvit86-urlshort-ms.herokuapp.com/SHORTENED_URL"}.
  • Passing an invalid URL will return {"error":"Invalid URL"}.
  • If, for example, the generated short URL is 2oefpk39takg4kkgcoc, visiting https://fabvit86-urlshort-ms.herokuapp.com/2oefpk39takg4kkgcoc will redirect to http://www.nba.com.
